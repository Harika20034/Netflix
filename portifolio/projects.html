Projects

Project 1

The Feedback Information System is a web-based application designed to collect, manage, and analyze feedback from users.

The system is built using a combination of Flask for the backend, HTML and CSS for the frontend, and JavaScript for interactivity and dynamic content.

This project aims to provide a seamless and intuitive platform for users to submit feedback and for administrators to view and analyze the collected data.

Project 2

The Task Management System is a comprehensive web application designed to help individuals and teams manage their tasks and projects efficiently.

The system utilizes Angular for the frontend, Flask (Python) for backend APIs, and Node.js for additional server-side functionalities.

The goal is to provide a user-friendly platform that allows users to create, assign, track, and complete tasks seamlessly.
